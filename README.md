# مدیریت کارها - Todo List (C++)

## معرفی پروژه

این پروژه برای درس **برنامه‌سازی پیشرفته** نوشته شده و هدفش تمرین مفاهیم شیءگرایی در ++C هست.
برنامه یک Todo List کنسولی است که کاربر می‌تواند کارهای روزانه خودش را با اولویت و ددلاین ثبت کند، پیگیری کند و موقع خروج همه چیز توی فایل ذخیره بشود.

---

## هدف پروژه

هدف اصلی این بود که مفاهیم زیر را عملی تمرین کنم:

- طراحی کلاس و ساخت شیء
- وراثت و چندریختی (توابع مجازی)
- سربارگذاری عملگرها
- کار با اشاره‌گرها و حافظه پویا
- خواندن و نوشتن فایل با `fstream`
- مرتب‌سازی با لامبدا و `std::sort`

---

## توضیح کلی برنامه

برنامه یک منوی متنی دارد که کاربر از طریق آن تسک‌هایش را مدیریت می‌کند.
هر تسک شامل عنوان، توضیحات، اولویت (کم/متوسط/زیاد) و ددلاین است.
برنامه تسک‌های منقضی‌شده را خودش تشخیص می‌دهد و تسک‌ها موقع بستن برنامه در فایل `tasks.txt` ذخیره می‌شوند تا دفعه بعد از بین نروند.

---

## امکانات برنامه

- افزودن تسک جدید (عادی یا ویژه)
- نمایش همه تسک‌ها
- نمایش تسک‌های انجام‌شده / در انتظار / منقضی‌شده
- علامت‌گذاری تسک به‌عنوان انجام‌شده
- حذف تسک با تأیید کاربر
- مرتب‌سازی بر اساس اولویت یا ددلاین
- ذخیره و بارگذاری خودکار از فایل

---

## توضیح بخش‌های مهم کد

### کلاس Date و Priority

کلاس `Date` تاریخ را مدیریت می‌کند و اعتبارسنجی تاریخ و تشخیص سال کبیسه را انجام می‌دهد.
کلاس `Priority` هم سطح اولویت را با مقدار ۱ تا ۳ نگه می‌دارد.
برای هر دو کلاس عملگرهای مقایسه و ورودی/خروجی سربارگذاری شده.

### نمایش تسک‌ها

از تابع مجازی `display` استفاده شده تا هر نوع تسک خروجی متفاوت خودش را داشته باشد.
فیلتر کردن تسک‌ها (انجام‌شده، در انتظار، منقضی) هم جداگانه پیاده‌سازی شده تا منطق نمایش از داده جدا بماند.

### ذخیره و بارگذاری فایل

تسک‌ها موقع خروج در `tasks.txt` نوشته می‌شوند و موقع اجرا دوباره خوانده می‌شوند.
نوع تسک (عادی یا ویژه) هم داخل فایل ذخیره می‌شود تا موقع بارگذاری درست بازسازی بشود.

### مرتب‌سازی

مرتب‌سازی با `std::sort` و لامبدا انجام شده و فقط اشاره‌گرها جابه‌جا می‌شوند نه خود داده‌ها.

## ساختار کلاس‌ها

```
Task (کلاس پایه)
 └── DeadlineTask (کلاس مشتق)

Date          → مدیریت و اعتبارسنجی تاریخ
Priority      → سطح اولویت (۱=کم  ۲=متوسط  ۳=زیاد)
TodoList      → مدیریت مجموعه تسک‌ها و فایل
```
